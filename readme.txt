# Simple HTML & CSS Calculator

This is a simple calculator project built with HTML and CSS. It's a straightforward web-based calculator that can perform basic arithmetic operations, such as addition, subtraction, multiplication, and division.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## Usage

To use the calculator, simply open the `index.html` file in your web browser. The calculator provides a user-friendly interface for entering numbers and performing calculations.

## How it Works

- Click on the calculator buttons to input numbers and operators.
- Click the equal (=) button to calculate the result.
- Click the clear (C) button to clear the input.
